Thermostat
==========

Remote Controlled Thermostat




Arduino sketch
==========

Currently implemented:

* Rotary Encoder
* Rotary Encoder RGB Led
* Rotary Encoder Switch
* 20x4 LCD
* LCD backlight control
* distance sensor (IR Led + photodiode)
* Light sensor
* relay output (uses external 2-channel relay block)
